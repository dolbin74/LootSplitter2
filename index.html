Logic Explanation

My loot array is the single source of truth. Every time loot is added or removed, the updateUI() function recalculates totals, updates the loot table, and controls what is visible or enabled. I added quantity to the data model so total loot is calculated as value × quantity. updateUI() performs all calculations and rendering so there is no duplicate logic. State controls the interface: when the loot array is empty, the table hides and the “No loot added” message appears. Conditional logic prevents invalid state by validating inputs before modifying the array.
